# Next.js SaaS starter

Reference template target for using agent-startup-kit skills.

## Intended stack
- Next.js
- TypeScript
- App Router
- Tailwind CSS
- Auth provider
- Billing provider

## Suggested use
- use ship-auth for authentication flows
- use ship-billing for pricing and subscription flows
- use ship-landing for landing page structure

## Goal

Provide a simple baseline project shape for demonstrating the skills.
