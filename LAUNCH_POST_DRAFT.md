## Slightly longer post

I just published `agent-startup-kit`, a lightweight open-source repo for reusable AI coding workflows.

The current v0.1 focus is simple:
- auth
- billing
- landing pages

The repo currently includes:
- skill directories with `README.md`, `skill.md`, and `checklist.md`
- before/after example docs
- Claude Code and VS Code adapter docs
- quickstart and index docs

The goal is not to build another agent framework.

The goal is to make repeated SaaS implementation tasks easier to structure, review, and reuse inside AI coding workflows.

Current repo entry points:
- `README.md`
- `QUICKSTART.md`
- `skills/README.md`
- `examples/README.md`
- `adapters/README.md`
- `templates/README.md`

This is an early skeleton release, but it is now public and ready for iteration.
