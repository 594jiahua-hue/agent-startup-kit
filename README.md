# agent-startup-kit

AI coding workflows and reusable skills for shipping SaaS fast.

Stop rebuilding the same auth, billing, landing, and deployment flows from scratch.

Install reusable skills for Claude Code and VS Code, then ship faster with consistent workflows.

## What this project does

agent-startup-kit helps indie hackers and small engineering teams use AI coding tools more effectively.

It provides reusable workflows, skill docs, checklists, and project structure for common SaaS building tasks.

## v1 focus

- ship-auth
- ship-billing
- ship-landing

## Supported platforms

- Claude Code
- VS Code

MCP support is planned for a later version.

## Why this exists

Most builders repeat the same setup work again and again:
- authentication
- billing
- landing pages
- deployment setup
- project structure
- workflow documentation

This project turns those repeated tasks into reusable AI coding workflows.
