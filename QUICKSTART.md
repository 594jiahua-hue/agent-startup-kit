# Quickstart

This guide shows the fastest way to start using agent-startup-kit.

## Step 1: Choose your tool

Current supported tools:
- Claude Code
- VS Code

## Step 2: Choose one skill

Start with one of these:
- `ship-auth`
- `ship-billing`
- `ship-landing`

## Step 3: Read the skill files

Inside each skill directory, read:
- `README.md`
- `skill.md`
- `checklist.md`

## Step 4: Review an example

Check the matching before/after example in:
- `examples/before-after-auth/`
- `examples/before-after-billing/`
- `examples/before-after-landing/`

## Step 5: Apply the workflow

Use the selected `skill.md` as working instructions in your AI coding workflow.

Then use the matching `checklist.md` to review the result.

## Suggested first path

If you are trying this project for the first time, start with:
1. `skills/ship-auth/`
2. `examples/before-after-auth/`
3. `adapters/claude-code/README.md` or `adapters/vscode/README.md`
